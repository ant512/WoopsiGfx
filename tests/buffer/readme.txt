Simple Picture
--------------

  This example shows illustrates some of the features of the WoopsiGfx library.
  It shows how to:
  
   - Create bitmap objects ready for drawing;
   - Use Graphics objects for drawing to bitmaps;
   - Use the FrameBuffer class to wrap a bitmap object around the DS'
     VRAM;
   - Initialise the DS' hardware ready for WoopsiGfx;
   - Copy a bitmap from one to another;
   - Use fonts to draw to a bitmap.
  
  
Building the Example
--------------------

  To build this example, open a shell, navigate to this directory, and type
  "make".
